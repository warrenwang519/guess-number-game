## Project Overview

A simple Python number guessing game. The player guesses a random number between 1-100, receiving "too high" or "too low" hints until they guess correctly.

## Running

```bash
python guess.py
```

This is an interactive program — it requires terminal input and cannot be run through non-interactive shells.

## Architecture

Single-file project: `guess.py` contains all game logic. No dependencies beyond Python standard library (`random`).
